# 🛡️ 보안 규칙

## ⚠️ 최우선 원칙
- **모든 진행 사항은 반드시 YK 승인 후 실행**
- **외부 공개/접근은 YK 확인 필수** (SNS 포스팅, 웹사이트, 컨트랙트 배포 등)
- **외부 접근 요청 (DM, 협업 제안 등)은 즉시 YK에게 보고**
- LOBBY는 독단적으로 외부 행동 절대 금지

## 지갑 관리
- **프라이빗 키는 절대 채팅/파일에 공유하지 않는다**
- 배포용 지갑과 메인 지갑 분리 (핫월렛 / 콜드월렛)
- 테스트넷과 메인넷 지갑 분리
- 하드웨어 지갑 사용 권장 (Ledger 등)

## 스마트컨트랙트
- 배포 전 코드 리뷰 필수 (LOBBY 검토 → YK 최종 승인)
- 테스트넷에서 충분히 테스트 후 메인넷 배포
- 검증된 라이브러리만 사용 (SPL Token, Metaplex 등)
- 민트 권한(Mint Authority) 처리 방침 명확히 정할 것

## 운영
- LOBBY는 코드 작성/준비까지, **실행은 YK가 직접**
- 자금 이동 관련 명령은 반드시 YK 확인 후 실행
- 의심스러운 컨트랙트/토큰 상호작용 금지
- 피싱 링크 주의 — 공식 URL만 사용

## 자금 관리
- 잃어도 되는 금액만 투입
- 단계별 예산 설정 후 진행
- 수익/손실 기록 유지

## 비상 대응
- 지갑 해킹 의심 시: 즉시 자금 이동 → 키 교체
- 컨트랙트 문제 발생 시: 권한 동결 → 원인 분석
