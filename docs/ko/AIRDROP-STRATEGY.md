## 4. 봇 방지 전략

### 4.1 등록 단계
- **CAPTCHA** (hCaptcha) 적용
- **트위터 계정 나이:** 30일 이상
- **트위터 팔로워:** 20명 이상
- **텔레그램:** 유저네임 설정됨, 14일 이상 계정
- **지갑 중복 제출 불가** (중복 = 실격)

### 4.2 행동 분석
- **IP 중복 제거:** IP당 최대 2건
- **지갑 클러스터링:** 같은 소스에서 24시간 내 펀딩된 지갑 플래그
- **트윗 분석:** AI로 복사-붙여넣기 감지 (원본이어야 함)
- **시간 분석:** 60초 미만 완료 = 봇으로 플래그

### 4.3 검증 레이어
- **랜덤 샘플링:** 10% 수동 검토
- **소셜 교차 확인:** 배분 시점에 팔로우/RT 유지 확인
- **지갑 활동 확인:** 1건 이상 SOL 트랜잭션 이력 필요

### 4.4 배분 후
- **시빌 감지:** 다수 에어드롭 지갑 → 동일 목적지 48시간 내 전송 시 플래그
- **커뮤니티 대시보드:** 에어드롭 지갑 행동 공개 추적
