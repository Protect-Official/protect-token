# 경쟁 분석 — 기부/자선 테마 밈코인 (Solana 중심)

*최종 업데이트: 2026-02-10*

---

## 1. 시장 개요

기부/자선 테마 밈코인은 틈새 시장이다. 대부분 BSC나 이더리움에서 런칭했으며, Solana 네이티브 자선 토큰은 거의 없다. **이것이 $PROTECT의 기회.**

**핵심 발견: 2026년 초 기준 Solana에서 지배적인 자선 밈코인은 존재하지 않는다.**

---

## 2. 주요 경쟁/유사 프로젝트

### 2.1 $CHILDREN / Save The Children Token (BSC)
- **체인:** BSC
- **런칭:** 2021년
- **컨셉:** Save the Children NGO에 기부하는 자선 토큰
- **최고 시가총액:** ~$30M
- **현재:** 사망 / 거래량 거의 0
- **성공 요인:** 강한 감정적 호소, 2021 BSC 밈 붐
- **실패 원인:** 지속적 커뮤니티 관리 실패, 팀 잠수, 투명성 보고 중단
- **교훈:** 지속적인 투명성과 정기적 기부 증명이 핵심

### 2.2 $GRUMPY (이더리움)
- **체인:** 이더리움
- **런칭:** 2021년
- **컨셉:** 고양이 밈 + 동물 보호소 기부
- **최고 시가총액:** ~$80M
- **실패 원인:** 높은 가스비로 소액 투자자 이탈, 기부 외 로드맵 부재
- **교훈:** 저렴한 수수료 체인(Solana ✓) 필수, 기부 외 유틸리티 필요

### 2.3 엡스타인 테마 토큰들 (Solana/멀티체인)
- **런칭:** 2023-2024년 (다수 카피캣)
- **최고 시가총액:** $1M-$10M
- **현재:** 대부분 며칠 내 사망
- **실패 원인:** 순수 투기, 기부 메커니즘 없음, 착취적으로 인식
- **교훈:** $PROTECT는 실제 아동보호 NGO 기부로 차별화 — 분노를 행동으로

### 2.4 $BONK 자선 활동 (Solana)
- 자선 토큰은 아니지만 커뮤니티 기부 드라이브 진행
- **의미:** Solana 밈코인 커뮤니티가 선한 목적을 위해 동원될 수 있음을 증명

## 4. 핵심 성공 요인

1. **투명성이 전부** — 기부 영수증 공개한 프로젝트가 더 오래 생존
2. **밈 에너지 ≠ 착취** — 대의가 힘을 주는 느낌이어야 함
3. **저렴한 수수료** — 가스비로 소액 기부자를 잃지 말 것
4. **커뮤니티 거버넌스** — 기부처 투표가 참여도를 높임
5. **정기적 업데이트** — 주간/월간 리포트로 신뢰 유지
6. **LP 잠금/소각** — 신뢰를 위한 필수 조건
7. **타이밍** — 뉴스 사이클과의 연계가 촉매제
8. **지속가능성** — 기부 외 유틸리티 필요 (거버넌스, NFT, 대시보드)
